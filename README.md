# Виртуальная экскурсия по институту — инструкция по запуску

Ниже — пошаговая инструкция, как запустить проект на своём компьютере.
Следуйте пунктам по порядку.

---

## 1. Что нужно установить заранее

Для запуска нужен только **Node.js** (версии 18 или новее; проект проверялся на Node 20).
Вместе с ним автоматически ставится менеджер пакетов **npm** — отдельно устанавливать его не нужно.

1. Откройте сайт <https://nodejs.org/> и скачайте версию **LTS**.
2. Установите её, соглашаясь со стандартными настройками.
3. Проверьте, что всё установилось. Откройте терминал и выполните:

   ```bash
   node --version
   npm --version
   ```

   Если в ответ выводятся номера версий (например `v20.20.1` и `10.8.2`) — всё готово.


---

## 2. Важно: медиафайлы (панорамы и видео)

360°-панорамы и видео **очень большие** и поэтому **не хранятся в репозитории GitHub**.
Без них проект запустится, но экскурсия будет пустой (чёрный экран вместо панорам).

Убедитесь, что в проекте есть папки с медиа:

```
public/assets/panoramas/   ← файлы .jpg (панорамы)
public/assets/videos/      ← файлы .mp4 (видео)
```

- **ZIP архив с public/assets можно скачать по ссылке** — https://disk.yandex.ru/d/xBScSGoifYcruA
---

## 3. Установка зависимостей

Откройте терминал **в папке проекта** 

> Например: 
`cd ~/Desktop/IKIT-VR-TOUR` (Linux/macOS) 
`cd C:\Users\Имя\Desktop\IKIT-VR-TOUR` (Windows).

Выполните команду (один раз, при первом запуске):

```bash
npm install
```

Она скачает все нужные библиотеки в папку `node_modules`. Это может занять пару минут.

---

## 4. Запуск проекта

Выполните:

```bash
npm run dev
```

В терминале появится строка вида:

```
  ➜  Local:   http://localhost:5173/
```

Откройте этот адрес **http://localhost:5173/** в браузере (Chrome, Firefox или Edge).
Экскурсия запустится.

Чтобы **остановить** проект — вернитесь в терминал и нажмите `Ctrl + C`.

---

## 5. Управление в экскурсии

- **Вращение обзора** — зажмите левую кнопку мыши и двигайте (на телефоне — проведите пальцем).
- **Приближение / отдаление** — колесо мыши (на телефоне — щипок двумя пальцами).
- **Переходы между точками** — кликните по стрелке/маркеру на панораме.
- **Мини-карта** — показывает все точки и ваше текущее положение.
- **Полноэкранный режим** — кнопка в верхней панели.
